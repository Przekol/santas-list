# ‚öôÔ∏èInstallation



```bash
# Clone this project
$ git clone https://github.com/przekol/santas-list

# Access
$ cd santas-list

# Install dependencies
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

#### The swagger  will initialize in the http://localhost:3000/api
